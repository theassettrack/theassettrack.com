# Asset Track Website

## Description

This project is a static website designed to present the Asset Track service. The site is built with pure HTML, CSS, and SVG, requires no backend logic, and does not need any build process.

## Project Structure

- `index.html` — Main landing page.
- `privacy-notice.html` — Privacy policy page.
- `files/` — Media, styles, and scripts:
  - `page_styles/` — Additional CSS styles.
  - `page_scripts/` — (Folder exists, but main.js is missing).
  - `video/` — Video files for the site.
- `assets/` — Graphic resources, icons, and fonts.

## Local Launch

To view the site locally:

1. Make sure you have Node.js installed.
2. In the project root, run:
   ```sh
   npx http-server ./ -p 8080
   ```
3. Open in your browser: [http://localhost:8080](http://localhost:8080)

## Features

- Fully static site, no backend required.
- Uses SVG illustrations and custom fonts.
- For correct resource loading, it is recommended to run via a local server.
