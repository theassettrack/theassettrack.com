# Asset Track — Статический сайт

## Описание

## Структура проекта

- `index.html` — Главная страница сайта.
- `privacy-notice.html` — Страница с политикой конфиденциальности.
- `files/` — Папка с медиафайлами, стилями и скриптами:
  - `page_styles/` — Дополнительные CSS-стили.
  - `page_scripts/` — (Папка существует, но основной файл main.js отсутствует).
  - `video/` — Видеофайлы для сайта.
- `assets/` — Графические ресурсы, иконки, шрифты.

## Запуск локально

Для локального просмотра сайта:

1. Убедитесь, что у вас установлен Node.js.
2. В корне проекта выполните команду:
   ```sh
   npx http-server ./ -p 8080
   ```
3. Откройте в браузере: [http://localhost:8080](http://localhost:8080)

## Особенности

- Сайт полностью статический, не требует backend.
- Используются SVG-иллюстрации и кастомные шрифты.
- Для корректной работы всех ресурсов рекомендуется запускать через локальный сервер.
